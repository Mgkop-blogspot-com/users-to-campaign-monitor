A plugin to push all new registered user's emails to Campaign Monitor.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/users-to-campaign-monitor` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Tools->Users to Campaign Monitor screen to add the "List ID" of your Campaign Monitor List.
4. Add a 'UTCM_USERNAME' constant to 'wp-config.php' with your API username key.
